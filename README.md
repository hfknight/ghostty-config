# Ghostty Config
My personal ghostty configuration on mac os

## Install 
1. copy config to ~/.config/ghostty/config or edit settings in Ghostty
2. Cmd + Shift + , to reload config
3. (optional) install [starship](https://starship.rs/) shell prompt
```bash
brew install starship
starship preset catppuccin-powerline -o ~/.config/starship.toml
```

## Common Commands
- Cmd + D → Add split view
- Cmd + T → New Tab
- Cmd + W → Close Tab/Surface
- Cmd + Shift + Enter → Toggle full screen
- Cmd + Q → Quit
- Cmd + Shift + , → Reload configuration
